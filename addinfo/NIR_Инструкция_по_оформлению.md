# ИНСТРУКЦИЯ ПО ОФОРМЛЕНИЮ НИР

## Общие требования

1. **Объем работы:** 20-30 страниц печатного текста
2. **Формат:** А4
3. **Шрифт:** Times New Roman, размер 14 пт
4. **Межстрочный интервал:** 1,5
5. **Поля:** 
   - левое - 30 мм
   - правое - 15 мм
   - верхнее - 20 мм
   - нижнее - 20 мм
6. **Абзацный отступ:** 1,25 см

## Структура работы

1. **Титульный лист** (1 страница)
2. **Содержание** (1 страница)
3. **Введение** (2-3 страницы)
4. **Основная часть** (15-20 страниц):
   - Раздел 1
   - Раздел 2
   - Раздел 3
5. **Заключение** (1-2 страницы)
6. **Список использованных источников** (1-2 страницы)
7. **Приложения** (при необходимости)

## Оформление заголовков

- **Заголовки разделов:** заглавные буквы, жирный шрифт, выравнивание по центру
- **Заголовки подразделов:** строчные буквы, жирный шрифт, выравнивание по левому краю
- **Нумерация:** арабскими цифрами (1, 1.1, 1.1.1)

## Оформление рисунков

- Рисунки должны иметь подписи под рисунком
- Формат: "Рисунок 1.1 - Название рисунка"
- Выравнивание по центру
- Ссылки в тексте: "(рисунок 1.1)"

## Оформление таблиц

- Таблицы должны иметь заголовки над таблицей
- Формат: "Таблица 1.1 - Название таблицы"
- Выравнивание по левому краю
- Ссылки в тексте: "(таблица 1.1)"

## Оформление формул

- Формулы выравниваются по центру
- Нумерация справа в скобках: (1.1)
- Ссылки в тексте: "формула (1.1)"

## Оформление списка литературы

- Нумерация арабскими цифрами
- Алфавитный порядок (по первой букве фамилии автора)
- Для книг: Фамилия, И. О. Название книги / И. О. Фамилия. - Город: Издательство, Год. - Количество страниц с.
- Для статей: Фамилия, И. О. Название статьи / И. О. Фамилия // Название журнала. - Год. - № Номер. - С. Страницы.
- Для интернет-источников: Название [Электронный ресурс]. - Режим доступа: URL (дата обращения: ДД.ММ.ГГГГ)

## Важные моменты

1. **Не используйте сложные технические термины** без объяснения
2. **Пишите простым языком** - работа должна быть понятна даже тем, кто не разбирается в программировании
3. **Делайте выводы** в конце каждого раздела
4. **Используйте примеры** для иллюстрации своих мыслей
5. **Ссылайтесь на источники** при упоминании фактов и данных

## Что нужно сделать

1. Скопировать текст из файла `NIR_Текст_работы.md` в Word
2. Отформатировать согласно требованиям
3. Добавить титульный лист из `NIR_Титульный_лист.md`
4. Добавить скриншоты интерфейсов платформ (Coursera, Udemy, Skillbox) в приложение
5. Проверить орфографию и пунктуацию
6. Распечатать и сшить работу

## Советы для защиты

1. **Подготовьте презентацию** на 5-7 слайдов:
   - Титульный слайд
   - Актуальность темы
   - Цель и задачи
   - Основные результаты исследования
   - Выводы

2. **Подготовьте ответы на возможные вопросы:**
   - Почему выбрана эта тема?
   - Какие методы исследования использовались?
   - Какие основные выводы?
   - В чем практическая значимость работы?

3. **Речь для защиты** (3-5 минут):
   - Представьтесь и назовите тему
   - Расскажите об актуальности
   - Опишите основные результаты
   - Сделайте выводы
