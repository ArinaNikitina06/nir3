# Быстрый старт

## Шаг 1: Установка зависимостей

```bash
npm install
```

## Шаг 2: Настройка базы данных

1. Создайте файл `.env` в корне проекта:
```env
DATABASE_URL="postgresql://user:password@localhost:5432/learning_platform"
NEXTAUTH_SECRET="your-secret-key-here"
NEXTAUTH_URL="http://localhost:3000"
```

2. Сгенерируйте Prisma Client:
```bash
npx prisma generate
```

3. Создайте миграции базы данных:
```bash
npx prisma migrate dev --name init
```

4. (Опционально) Заполните базу тестовыми данными:
```bash
npx prisma db seed
```

## Шаг 3: Запуск приложения

```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Тестовый аккаунт

## Основные команды

- `npm run dev` - запуск в режиме разработки
- `npm run build` - сборка для продакшена
- `npm start` - запуск продакшен версии
- `npm run lint` - проверка кода линтером
- `npx prisma studio` - открыть Prisma Studio для просмотра БД

## Структура базы данных

После выполнения миграций будет создана база данных со следующими таблицами:
- `User` - пользователи
- `Course` - курсы
- `Lesson` - уроки
- `Enrollment` - записи на курсы
- `Progress` - прогресс обучения
- `UserPreference` - предпочтения пользователей
- `UserInteraction` - история взаимодействий

## Настройка предпочтений

2. Перейдите в профиль (`/profile`)

4. Сохраните изменения
