# Описание проекта для ВКР

## Тема работы
**Исследование UX-подходов и разработка веб-интерфейса персонализированных рекомендаций для онлайн-платформы обучения на Next.js с целью повышения удобства выбора курсов**

## Цель проекта
Разработать веб-приложение онлайн-платформы обучения с интеллектуальной системой персонализированных рекомендаций курсов, основанной на современных UX-подходах и алгоритмах машинного обучения.

## Основные задачи

1. **Исследование UX-подходов** для персонализированных рекомендаций в образовательных платформах
2. **Разработка алгоритма рекомендаций** на основе многофакторного анализа
3. **Создание современного веб-интерфейса** с использованием Next.js и Tailwind CSS
4. **Реализация системы отслеживания** пользовательских взаимодействий и предпочтений
5. **Оптимизация пользовательского опыта** при выборе и записи на курсы

## Ключевые особенности

### 1. Алгоритм персонализированных рекомендаций

Система использует многофакторный подход для расчета релевантности курсов:

- **Предпочтения пользователя** (30 баллов) - соответствие выбранной категории и уровню подготовки
- **История взаимодействий** (20 баллов) - анализ просмотров, лайков, закладок
- **Популярность курса** (15 баллов) - количество студентов и рейтинг
- **Соответствие интересам** (10 баллов) - совпадение тегов курса с интересами пользователя
- **Схожесть с просмотренными** (10 баллов) - рекомендации на основе похожих курсов

### 2. UX-подходы

#### Персонализация
- Индивидуальные рекомендации для каждого пользователя
- Объяснение причин рекомендаций (прозрачность алгоритма)
- Адаптация под предпочтения и уровень пользователя

#### Визуальная иерархия
- Четкое разделение топ-рекомендаций и популярных курсов
- Использование бейджей и индикаторов для выделения важной информации
- Цветовое кодирование категорий и уровней

#### Микро-интеракции
- Плавные анимации при загрузке и взаимодействии
- Hover-эффекты на карточках курсов
- Визуальная обратная связь при действиях пользователя

#### Адаптивность
- Полностью адаптивный дизайн для всех устройств
- Оптимизация для мобильных, планшетов и десктопов
- Удобная навигация на любом экране

### 3. Технические решения

#### Frontend
- **Next.js 14** - современный React-фреймворк с App Router
- **TypeScript** - типобезопасность и лучшая разработка
- **Tailwind CSS** - утилитарный CSS-фреймворк для быстрой разработки
- **Framer Motion** - библиотека для плавных анимаций
- **Lucide React** - современные иконки

#### Backend
- **Next.js API Routes** - серверная логика
- **Prisma ORM** - типобезопасная работа с базой данных
- **PostgreSQL** - надежная реляционная база данных
- **NextAuth.js** - аутентификация и авторизация

#### База данных
- Модели для пользователей, курсов, уроков
- Система отслеживания прогресса обучения
- Хранение пользовательских предпочтений
- История взаимодействий для анализа
