# Онлайн-платформа обучения с персонализированными рекомендациями

Веб-приложение для онлайн-обучения, разработанное на Next.js с системой персонализированных рекомендаций курсов на основе UX-подходов.

## Описание проекта

Данный проект представляет собой полнофункциональную платформу онлайн-обучения с интеллектуальной системой рекомендаций, которая помогает пользователям находить наиболее подходящие курсы на основе их предпочтений, истории взаимодействий и прогресса обучения.

## Основные возможности

- 🔐 **Аутентификация и авторизация** - безопасная система входа и регистрации
- 🎯 **Персонализированные рекомендации** - умный алгоритм подбора курсов
- 📊 **Отслеживание прогресса** - мониторинг обучения пользователей
- 🎨 **Современный UX-дизайн** - интуитивный и удобный интерфейс
- 📱 **Адаптивный дизайн** - поддержка всех устройств
- 🔍 **Система взаимодействий** - отслеживание просмотров, лайков, закладок

## Технологический стек

- **Frontend**: Next.js 14, React 18, TypeScript
- **Styling**: Tailwind CSS
- **Database**: PostgreSQL, Prisma ORM
- **Authentication**: NextAuth.js
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Установка и запуск

### Требования

- Node.js 18+ 
- PostgreSQL
- npm или yarn

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd NIR
```

2. Установите зависимости:
```bash
npm install
```

3. Настройте переменные окружения:
Создайте файл `.env` в корне проекта:
```env
DATABASE_URL="postgresql://user:password@localhost:5432/learning_platform"
NEXTAUTH_SECRET="your-secret-key-here"
NEXTAUTH_URL="http://localhost:3000"
```

4. Настройте базу данных:
```bash
npx prisma generate
npx prisma migrate dev --name init
```

5. (Опционально) Заполните базу тестовыми данными:
```bash
npx prisma db seed
```

6. Запустите сервер разработки:
```bash
npm run dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

## Алгоритм рекомендаций

Система рекомендаций использует многофакторный подход:

1. **Предпочтения пользователя** (30 баллов) - соответствие категории и уровню
2. **История взаимодействий** (20 баллов) - просмотры, лайки, закладки
3. **Популярность курса** (15 баллов) - количество студентов и рейтинг
4. **Интересы пользователя** (10 баллов) - соответствие тегам
5. **Схожесть с просмотренными** (10 баллов) - рекомендации на основе похожих курсов

## UX-подходы

- **Персонализация** - индивидуальные рекомендации для каждого пользователя
- **Визуальная иерархия** - четкое разделение топ-рекомендаций и популярных курсов
- **Прозрачность** - объяснение причин рекомендаций
- **Адаптивность** - оптимизация для всех устройств
- **Микро-интеракции** - плавные анимации и переходы
